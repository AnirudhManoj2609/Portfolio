OCD 3D World

Project Overview

A multiplayer 3D world built with Unity for individuals with social anxiety to interact and explore in a safe environment. Includes avatar interaction, dynamic scene changes, and real-time chat using Spring Boot WebSockets.

Technology Stack
  • Unity (C#) for 3D simulation
  • Spring Boot for backend + WebSocket server
  • NativeWebSocket for Unity client networking
  • Docker, PostgreSQL, AWS for deployment
Key Features
  • Dynamic avatar movement & sync
  • Custom scene transitions
  • Secure login & profile avatars
Planned Improvements
  • Emotion-based avatars using ML
  • Voice chat support
  • AI chatbot NPCs for guided therapy

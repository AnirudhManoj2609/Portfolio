AAKA – Machine Learning Library

Project Overview

AAKA is a C++ library focused on optimizing machine learning workflows. It includes mathematical tools, training loops, optimizers, and data pre-processing utilities for building fast, low-level ML models.

Technology Stack
  • C++ STL, Templates, and Multi-threading
  • Custom numerical libraries for matrix operations
  • Cross-platform CLI and optional Python bindings
Core Features
  • Data loaders and shuffling
  • Gradient Descent, Adam, and RMSProp implementations
  • Serialization for model weights

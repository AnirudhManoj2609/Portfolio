MS-DOS Based Operating System

Project Overview

A bootable operating system written in x86 assembly and C. Inspired by MS-DOS, it features a simple command shell, memory management, and low-level BIOS interrupt handling.

Technology Stack
  • NASM for bootloader + kernel
  • C for I/O & shell
  • QEMU + Bochs for emulation
Highlights
  • Boots from MBR (512 bytes)
  • Supports basic shell commands
  • Loads and executes binary files
